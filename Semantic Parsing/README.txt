Steps to execute the program :-

1) Open terminal and go to the directory where you have this folder.

2) In terminal type $flex lex.l.
This will generate one .c file named as lex.yy.c

3) Then compile the newly formed lex.yy.c using $gcc lex.yy.c

4) Now run the program by &./a.out

5) After typing ./a.out you'll be promoted with enter any string . Type the string and program will generate all the tokens.
